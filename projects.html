Servo Peripheral Controller in VHDL

VHDL FPGA PWM Intel Quartus

Built a four-channel PWM servo controller implemented as a custom FPGA peripheral in VHDL. Features 7-bit position control (128 positions) with sub-microsecond timing accuracy using a PLL-derived 71 kHz clock. The design uses double-buffered position updates for glitch-free operation and memory-mapped I/O interface.

64-Bit Calculator Verification

SystemVerilog UVM Verification Cadence

Comprehensive SystemVerilog-based verification environment for a 64-bit calculator design. Implemented FSM-based controller with result buffering. Developed UVM-style testbench with driver, monitor, sequencer, and scoreboard. Includes SystemVerilog Assertions (SVA) and 200+ randomized test cases.

ADCS Flight Computer for Georgia Tech Rocketry

Altium PCB Design Embedded C Kalman Filter

Designed and routed the Attitude Determination and Control Systems board for the first collegiate two-stage rocket targeting the Kármán line. Features advanced sensors, robust data logging, and streamlined layout for precise attitude determination and reliable in-flight telemetry. Serves as the brain of the rocket with extended Kalman Filter for state estimation.
