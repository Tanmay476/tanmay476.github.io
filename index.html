About Me

I'm a Computer Engineering student at Georgia Tech specializing in digital design, VLSI, and embedded systems. I build verification-first hardware that scales from FPGA prototypes to tapeout-ready silicon.

At Georgia Tech's SHARC Lab, I contribute to HLSFactory, an open-source HLS→RTL automation framework that applies ML models to optimize latency, power, and area. I expanded the dataset with 100+ open-source designs and validated them via Vitis HLS, enabling 20% faster synthesis throughput versus naive parallelism on 32 CPU cores.

As a Design Verification Engineer with SiliconJackets, I develop SystemVerilog UVM testbenches for tapeout-bound processors. I built a comprehensive UVM environment for a Reorder Buffer (ROB) in an out-of-order RISC-V core, validating allocation, forwarding, and in-order retirement across parameterized widths with 99.5% code coverage.

As Avionics Hardware Lead at Georgia Tech Experimental Rocketry, I architect PC104 embedded systems for a two-stage rocket targeting the Kármán line. I designed the ADCS flight computer PCB in Altium, integrating high-speed sensors, robust power distribution, and telemetry. I co-developed 1,000+ lines of embedded C and Linux code implementing an Extended Kalman Filter for real-time state estimation.

I'm energized by the intersection of hardware and software: designing RTL, routing PCBs for aerospace applications, and optimizing chip layouts for performance. I'm seeking opportunities in semiconductor design, FPGA development, and hardware verification.

Impact Highlights

  • 99.5% verification code coverage for an out-of-order RISC-V ROB using UVM.
